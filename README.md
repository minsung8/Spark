

1. Apache Spark 란 ?

---------------------------------------------------------

2. Apache Spark vs Map Reduce
    - Map Reduce
        - 디스크에서 데이터를 처리 => 데이터 처리 속도가 느림 vs 디스크 용량만큼 한번에 처리 가능
    - Apache Spark
        -  메모리에서 데이터를 처리 => 데이터 처리 속도가 빠름 vs 메모리 용량만큼만 처리 가능 (부하에 불리)
        - 인메모리상에서 동작하기 때문에 반복처리에 유리
        - 실시간 데이터 처리에 유리
